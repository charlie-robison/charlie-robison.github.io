### A Problem Becomes an Idea

A major passion of mine since I was little is making music. I currently make Hip Hop beats and sell them to rappers for them to rap over. I however had a problem. I was having trouble finding rappers to work with and collaborate with. With that said, I decided to create an iPhone app that connects producers such as myself and rappers together using a matching feature. The app works by the user signing up and specifying what they are (producer, rapper, singer, etc). Then the user is able to go to a tab called match and they can select the type of musician they want to work with. Based on the selection the user makes, it displays all users under the given search selection. The user is then able to follow the person. Users are also able to post their music, and like/comment on people's music. 

### Development of RapCollab

I created the frontend of this app using Xcode with the swift programming language and UI. I built the backend using DJango with the python programming language and postgreSQL as the database. During this process I learned a lot about building apps, specifically building a backend and making API calls from the front end to receive and post data to and from the backend. This was an individual personal project for me so I took on the project learning all aspects of iOS development.

### The Knowledge Obtained from Development

Throughout this experience I learned DJango which taught me about the model-view way of making post, and get requests with the database. I also learned about serializers, which converted data from the database to JSON and vice-a-versa. I also learned a lot about UI in the frontend of the app. I learned how to connect UI to the code so that the app was functional such as inserting user info or making a post on their account. My final take away of this project is that it taught me how to develop large full-stack apps and how to integrate a backend and a frontend together.
