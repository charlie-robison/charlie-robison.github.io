Throughout my life so far I have always found joy in creating things. Once I learned about programming and computer science in high school, I grew a passion for creating software for fun ranging from video games to iOS applications. This is why I decided to take ICS 314, Introduction to Software Engineering. I know this class will be one of those special classes that will impact my life because it will teach me how to develop software in a professional setting. 

Main Interests In Software Engineering

Every aspect of Software Engineering interests me. However, my main interest in software engineering is learning how to put applications together that involve many different parts such as user interface (UI), client-side functionality, and server-side functionality aspects of the application. Over the summer I have been working on putting these parts together for iOS applications, however, I want to be able to learn how to do this in a professional setting and deviate towards applying this to web applications. With that being said, another interest of mine is learning more about web development. Since I already make iOS applications for fun, I would also like to expand my knowledge of web applications and even connect the two when appropriate. I feel Software Engineering will lead me to better understand my interests and form new interests.

Skills and Experiences I Would Like to Develop

I know I will develop many skills in this course but the main skills and experiences I want to take away from this are developing software in a professional setting, learning to collaborate with a team when developing software, and understanding new knowledge in web development. Developing software in a professional environment is an important experience to me because this is what I want my future to be. This also goes with learning how to collaborate with others in developing software. I want to learn to work with others on this so that I’ll be able to do that effectively later in my career. Lastly, I would love to gain more knowledge and insight on web development because I feel like this is a very important field of software and I am very curious to learn it.

Final Thoughts

In all, my main goal at the end of this is to become a professional developer who develops different types of software ranging from websites, video games, iOS applications, etc. This course will put me on the right path to achieve my main goal.
